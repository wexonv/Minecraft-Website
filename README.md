## About The Project

[![Minecraft Server Website Screenshot](http://wexon.top/repo/minecraft-website-view.png)](https://minecraft-website-one.vercel.app/)

A beautifully designed, fully responsive website template for Minecraft server owners. Showcase your server status, player count, and important links in a professional and modern interface.

Here's why this template is perfect for your Minecraft server:
* **Real-time Status**: Live server status checking with automatic updates
* **Player-Friendly**: Copy server IP with one click
* **Fully Customizable**: 6 color themes and easy configuration
* **Responsive Design**: Looks great on desktop, tablet, and mobile
* **SEO Optimized**: Better visibility on search engines
* **Fast & Lightweight**: Built with modern technologies for optimal performance

Perfect for server owners who want a professional web presence without the hassle of building from scratch.

## Getting Started

Get your Minecraft server website up and running in just a few minutes!

### Prerequisites

Before you begin, make sure you have:
* Node.js 18+ installed
* npm (comes with Node.js)
* A Minecraft server with a public IP
